StrTabs : module
sort by:oopabc

Imports: OS, Hashes, StrUtils

The strtabs module implements an efficient hash table that is a mapping from strings to strings. Supports a case-sensitive, case-insensitive and style-insensitive mode. An efficient string substitution operator % for the string table is also provided. Example:

import StrTabs

# allocate table to store input
var input = new StringTable

# add every line of stdin to the table
for idx, line in stdin.lines:
  input["ln-" & $(idx + 1)] = line

# print out table contents
for key, val in input:
  echo key, " : ", val
